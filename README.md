RubyChina for Chrome
======================

RubyChina for Chrome 是对 http://ruby-china.org 进行增强的 Google Chrome 扩展。

## 功能列表

 * 鼠标移动到会员的头像、名字上时，浮动显示会员的详细信息及最近话题
 * 发新帖和回复的表单支持全屏编辑功能
 * 在扩展图标上显示未读通知的数量，点击后打开通知页面
